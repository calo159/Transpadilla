# @workspace/api-types — Tipos Zod (generados)

Esquemas Zod y tipos TypeScript generados con orval a partir de
[`@workspace/api-spec`](../api-spec) (`openapi.yaml`). Los comparten frontend y backend
para validar/typar las entradas y salidas del API.

## ⚠️ No editar a mano
`src/` es **generado**. Para cambiarlo, edita la spec y regenera:
```bash
pnpm --filter @workspace/api-spec run codegen
```
